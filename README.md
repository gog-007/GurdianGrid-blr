# 🛡️ GuardianGrid: Bangalore Smart Mobility

GuardianGrid is a lightweight, local-first safety escort designed for the unique urban landscape of Bangalore. It prioritizes offline-first security logic to protect commuters during late-night travel.

### 📍 Bangalore Safety Matrix
Unlike standard GPS apps, GuardianGrid uses a **Local Safety Grid** of Bangalore-specific havens:
- **Namma Clinics** (Medical Safety)
- **Police Chowkys** (Security)
- **24/7 Tech Hubs** (Safe Lighting & Surveillance)

### 🧠 Features
- **Temporal Risk Engine:** Automatically escalates UI alerts based on distance from known safe zones in Bangalore.
- **Privacy-First:** Zero data leaves the device. All Haversine distance calculations are done locally in `script.js`.
- **Ultra-Lightweight:** Total repository size is < 200KB.

### 🚀 Tech Stack
- **Frontend:** HTML5 / Tailwind CSS
- **Logic:** Local-First Vanilla JS
- **Deployment:** Google Cloud App Engine
